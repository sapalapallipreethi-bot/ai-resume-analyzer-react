# AI Resume Analyzer

AI Resume Analyzer is a web application built using React and Vite that helps users upload their resumes, view an ATS score, and receive resume improvement suggestions through a clean and responsive interface.

## Features

- Resume Upload
- ATS Score Display
- Resume Improvement Suggestions
- Responsive User Interface
- Modern Design

## Technologies Used

- React
- Vite
- JavaScript
- HTML
- CSS

## Project Status

Frontend completed successfully.

Backend integration and advanced AI-powered resume analysis features are currently under development.
